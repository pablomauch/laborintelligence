# LaborCompliance Platform — BPS

Mockup funcional de la plataforma integral de Labor Compliance para BPS.

## Descripción

Plataforma de gestión de compliance laboral que cubre el ciclo de vida completo del contrato entre empresa mandante (ej. BHP Minera Escondida) y sus empresas contratistas y subcontratistas.

## Módulos

| Módulo | Descripción |
|--------|-------------|
| **Acreditación** | Gestión documental de ingreso, credenciales, alertas de vencimiento |
| **Auditoría Remuneracional** | Control mensual de pagos, cotizaciones, F30-1, ISC |
| **Auditoría Laboral & HSE** | Compliance normativo semestral, DS594, RIOHS, políticas mandante |
| **Inteligencia Laboral** | Análisis de instrumentos colectivos, mapa sindical, alertas |
| **Cierre de Contrato** | Cruce finiquitos vs depósitos, valorización incumplimientos |
| **Pagos & Juicios** | Bot Poder Judicial, subrogación, gestión causas |

## Características

- 🤖 **Agentes de IA** por módulo con procesamiento automático
- 📂 **Upload de documentos** con clasificación automática por IA
- 💬 **Chatbot asistente** con acceso a todos los módulos
- 🔗 **Integración entre módulos** — datos cruzados en tiempo real
- 📊 **Reportabilidad automática** consolidada
- 🚨 **Sistema de alertas** unificado

## Uso

Abrir `index.html` directamente en el navegador. No requiere servidor.

## Stack

- HTML5 / CSS3 / Vanilla JavaScript
- Google Fonts (DM Sans + DM Mono)
- Sin dependencias externas
